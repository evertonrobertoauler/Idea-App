Idea-App
========

A Angular.js app with a bunch of good ideas

## You'll need to install Node.js and Yeoman

http://nodejs.org/

http://yeoman.io/

## To intall the dependencies, run de folowing commands

        git clone https://github.com/evertonrobertoauler/Idea-App.git
        cd Idea-App
        npm install
        bower install
        
## To run the app

        grunt serve
